# Template Categories

Use category files in this folder to map cleaned templates to retrieval groups.

## Suggested Category Set

- `site-context-verification`
- `observed-condition-summary`
- `repair-remediation-recommendation`
- `permit-authority-support`
- `inspection-follow-up`
- `scope-limitation-partial-observation`

For each category, include:

- intent and non-intent
- required placeholders
- optional placeholders
- assumptions
- limitations
- wording to avoid
