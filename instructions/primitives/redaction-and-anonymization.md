# Redaction and Anonymization Primitive

Before extracting reusable language from a source letter:

1. Remove direct identifiers (names, exact addresses, contact details).
2. Remove project-unique identifiers unless structurally necessary.
3. Convert remaining specifics into placeholders.
4. Preserve only language patterns needed for scope, assumptions, limitations, and observations.

If uncertain whether text is identifying, treat it as sensitive and redact.
